What is this service?

This services enables you to find an NHS login user account and then perform the following actions depending on whether you have permission to perform them:

  • Disable the account - so that the user can no longer log in.
  • Cleanse the account - so that the account is no longer associated with any demographic details and the proofing level goes down to 0.
